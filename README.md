# Le Bonnet

> This is a modern ecommerce concept app - clone of the original one by me.

## Features

- Authentification system
- Shopify
- Cart
- Automatic notification

## Tech Stack

- ReactJS
- React hooks
- React dom
- React router
- Redux
- Redux-Thunk
- JWT
- Axios
- Styled-Components
- LazyLoading
- Firebase cloud functions
- Firebase firestore
  - NodeJS
  - Express
