# Le Bonnet

> This is a modern ecommerce concept app - clone of the original one by me.

## Features

- Authentication system
- Cart
- Automatic notification system

## Tech stack

- ReactJS
- React hooks
- React dom
- React router
- Redux
- Redux-Thunk
- JWT
- Axios
- Styled-Components
- LazyLoading
- Firebase cloud functions
  - NodeJS
  - Express
- Firebase firestore
